# Ontpluggen

React, TypeScript, Vite, and Tailwind CSS app for Ontpluggen.

## Requirements

- Node.js 18 or newer
- npm
- Git

## Get The Code

If this is your first time working on the project, clone the repository:

```bash
git clone https://github.com/azietipsnl-lab/ontpluggen.git
cd ontpluggen
```

If you already have the project locally, pull the latest changes:

```bash
git pull
```

## Install Dependencies

### Windows PowerShell

```powershell
npm install
```

If PowerShell blocks `npm` with a script execution policy error, use:

```powershell
npm.cmd install
```

### macOS Terminal

```bash
npm install
```

## Run Locally

### Windows PowerShell

```powershell
npm run dev
```

If PowerShell blocks `npm`, use:

```powershell
npm.cmd run dev
```

### macOS Terminal

```bash
npm run dev
```

Vite will print a local URL, usually:

```text
http://localhost:5173/
```

Open that URL in your browser.

## Useful Commands

Run a TypeScript check:

```bash
npm run typecheck
```

Create a production build:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
